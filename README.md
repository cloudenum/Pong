# Pong
Pong 2D Game  

A Game Project for DILo Game Academy 

Built using Unity 2018.3.3f1

## Controls
- ```W``` move up Player 1
- ```S``` move down Player 1
- ```Up Arrow``` move up Player 2
- ```Down Arrow``` move down Player 2
- ```F12``` Toggle debug info
